What I'm learning: HTML structure, basic tags, and how to style pages with CSS.

HTML Basics I've Learned

  • DOCTYPE: Tells the browser this is an HTML document
  • html tag: The root element of the page
  • head section: Contains metadata and styling
  • body section: Contains the visible content
  • h1, h2: Heading tags for titles
  • p: Paragraph tags for text
  • ul, li: Unordered lists and list items
